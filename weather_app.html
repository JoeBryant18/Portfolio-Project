Weather App

Description

A weather app that can get the temperature and humidity of the city provided in the search bar. It gets this information through a weather API by using an API key to fetch the data from the database. An emoji is displayed depending on the weather. If it is sunny, the emoji will be a sun. If it is cloudy it will be a cloud emoji.

Tools Used

  • HTML
  • CSS
  • JavaScript
  • VS Code
  • Windows 10
